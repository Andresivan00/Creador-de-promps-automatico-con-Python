# Creador de promps automatico (Python & Tkinter)

App de diseño e implementación de una aplicación 
gráfica creada en Python mediante la librería Tkinter. El propósito principal del proyecto es 
permitir al usuario generar prompts coherentes y descriptivos, basados en una serie de categorías, 
estilos e iluminaciones predefinidas; estos prompts son útiles para herramientas de generación de 
imágenes mediante inteligencia artificial. 
La aplicación organiza las opciones de manera visual y dinámica, validando las selecciones del 
usuario y construyendo un prompt final que puede ser copiado al portapapeles.

**Objetivo General:** Desarrollar una aplicación interactiva de escritorio que permita generar 
prompts detallados y coherentes para modelos de inteligencia artificial visual, combinando 
categorías, estilos, iluminación y composición de manera dinámica, proporcionando al usuario 
una herramienta intuitiva para la creación de contenido creativo de alta calidad. 

**Objetivos Específicos:**

• Diseñar una interfaz gráfica intuitiva y amigable que facilite la selección de categorías, 
elementos, estilos, iluminación y composición. 

• Implementar la lógica de generación de prompts adaptativos, que combine las opciones 
del usuario con descripciones detalladas y coherentes. 

• Validar las selecciones del usuario para garantizar que todos los campos estén completos 
antes de generar el prompt. 

• Incorporar funcionalidades de copia al portapapeles y visualización dinámica del prompt 
generado.
